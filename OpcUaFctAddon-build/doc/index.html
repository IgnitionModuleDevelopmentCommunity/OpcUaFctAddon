Instructions :

    Use the following functions in your client scoped scripts :

    For OPC tags subscriptions based on your dataset with TagFullPath, Value, LastChange and Quality columns, and update of a client tag dataset named [client]opcua_subscription you have to create before

  • byes.opcuafctaddon.subscribe(Dataset)
  • byes.opcuafctaddon.unsubscribeAll
  • byes.opcuafctaddon.freezeAll
  • byes.opcuafctaddon.unFreezeAll

Details for byes.opcuafctaddon.subscribe(Dataset):

    Subscribe directly to OPC tags change inside a dataset and update a client dataset. (Example of full OPC path ns=3:s=Liaison_TJO.DB2046.4,i)

    All columns beginning with 'OPC_' are managed by this module function addon.
    Group of column must have at least a column '.TagFullPath' property.
    It could me more that one group in a datasetProperties '.Value','.Quality','.LastChange' are optional

    Example columns name in dataset: A,OPC_X.TagFullPath,OPC_Y.TagFullPath,OPC_X.Value,B,OPC_X.Quality,OPC_Y.Value
    • There is 2 groups of column manage -> X and Y
    • The column A and B are not manage and stay unchanged
    • OPC_X.TagFullPath -> list of all full OPC tags group X | OPC_Y.TagFullPath -> list of all full OPC tags group Y
    • OPC_X.Value -> a column contain value group X | OPC_Y.Value -> a column contain value group Y
    • OPC_X.Quality -> a column contain quality group X
